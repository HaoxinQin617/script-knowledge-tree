# 主页“讲解图”分类设计

## 目标

在主页增加“讲解图”分类，让用户能够按口播主题快速浏览讲解图片，并直接进入对应口播页面查看完整内容。

## 主页分类

主页分类调整为：

- 全部
- 定义
- 使用 AI
- 讲解图

“讲解图”分类覆盖第一、第二、第三层所有拥有多页举例图集的口播，不受原有首页只展示第一层主题的限制。

## 讲解图卡片

每个口播只显示一张卡片：

- 封面固定使用该口播“对应的多页举例图集”的第一页。
- 显示口播标题、所属层级、预计口播时长和图集页数。
- 保留搜索、日期筛选与日期排序。
- 卡片图片使用完整比例显示，避免裁切文字或逻辑关系。

## 点击行为

点击封面或卡片后，进入现有的对应口播页面，不创建独立图片详情页。

进入口播页面后，用户仍可查看：

- 原始首图；
- 两张结构图预览；
- 完整的多页举例图集；
- 口播正文和当前层级的上下级导航。

## 空状态与兼容性

- 没有多页举例图集的口播不出现在“讲解图”分类。
- 搜索或日期筛选后没有结果时，继续使用现有主页空状态。
- 桌面端与移动端共用同一数据来源；移动端卡片保持单列和可触控区域。
- 不改变现有口播文字、生成方式、层级关系或其他分类的行为。

## 验收标准

1. 主页出现“讲解图”分类按钮。
2. 该分类列出所有层级中拥有图集的口播。
3. 每张卡片封面均为对应图集第一页。
4. 点击卡片进入正确的口播页面。
5. 口播页面中的完整图集仍可翻阅。
6. 搜索、日期筛选和排序正常工作。
7. 桌面端与移动端均不会裁切封面中的主要文字与关系线。
