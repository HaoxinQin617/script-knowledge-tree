# Design Systems Catalogue
## Reference catalogue derived from alexpate/awesome-design-systems

For full 9-section profiles, see individual files in `./profiles/`.
This catalogue provides compact signal attributes for every system — enough for an AI agent to generate an informed DESIGN.md inspiration pass.

---

## Format
Each entry covers:
- **URL** — source
- **Archetype** — `developer-minimal` | `enterprise-data` | `premium-editorial` | `playful-gradient` | `cinematic-dark` | `civic-accessible` | `media-forward`
- **Colors** — primary brand hex + notable palette notes
- **Typography** — key typefaces
- **Density** — airy / balanced / dense
- **Notable for** — one-line differentiator

## Style Archetypes Quick Reference
| Archetype | Characteristics | Examples |
|---|---|---|
| `developer-minimal` | Monochrome or neutral, code-forward, dark mode first | GitHub Primer, shadcn, Vercel Geist, Carbon |
| `enterprise-data` | Dense, systematic, multi-column, semantic color | IBM Carbon, Atlassian, SAP Fiori, AWS Cloudscape |
| `premium-editorial` | Large typography, whitespace, photography | Apple HIG, Artsy, FT Origami, Porsche |
| `playful-gradient` | Vibrant, rounded, illustrated, animated | Material M3, Duolingo, Vibe, Mailchimp |
| `cinematic-dark` | Dramatic contrast, glow restraint, full-bleed imagery | Warp, RunwayML, ElevenLabs |
| `civic-accessible` | High contrast, flat, accessible, institutional | GOV.UK, USWDS, Aurora Canada, NHS |
| `media-forward` | Image-first, editorial typography, discovery grid | Pinterest Gestalt, BBC GEL, Yelp, FT |
